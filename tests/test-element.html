Cui Menu (새 수직 메뉴)

cui-menu는 cui-nav-tree를 대체하는 새로운 수직 메뉴 컴포넌트입니다.

클릭 모드 (기본)

  • dashboard 대시보드
  • shopping_cart 주문관리
    • list_alt 주문 목록
    • local_shipping 배송 관리
      • 국내 배송
      • 해외 배송
      • 배송 추적
    • undo 환불/취소
      • 환불 신청
      • 취소 내역
  • inventory 상품관리
    • view_list 상품 목록
    • category 카테고리
      • 대분류
      • 중분류
      • 소분류
  • settings 설정
