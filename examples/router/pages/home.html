특징

  • cui-href - 링크 클릭 시 페이지 로드
  • cui-target-content - 페이지 렌더링 영역
  • 페이지별 모듈 로드 - Catui.use로 필요한 모듈만 로드
  • 인스턴스 관리 - 페이지 이동 시 자동 정리 (메모리 누수 방지)

빠른 이동

폼 예제 테이블 예제 팝업 예제 컴포넌트 타이머 (인스턴스 관리)

기본 API

메서드 설명
router.go(path)페이지 이동
router.back()뒤로 가기

router.current()현재 라우트 정보
