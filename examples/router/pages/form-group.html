폼 그룹 클래스 요약

    

      

        
          

            클래스
            설명
          

        
        
          
.cui-form폼 컨테이너

          
.cui-form-item폼 항목 (라벨 + 입력)

          
.cui-form-label폼 라벨

          
.cui-form-input폼 입력 영역

          
.cui-form-tip폼 도움말

          
.cui-form-inline인라인 폼

          
.cui-form-danger검증 실패 상태

          
.cui-form-success검증 성공 상태

        
      

    

  

  
  
  

    
폼 속성 요약

    

      

        
          

            속성
            설명
          

        
        
          
cui-filter폼 식별자 (form.on 이벤트에서 사용)

          
cui-verify검증 규칙 (required, email, phone, pass, url, number, date)

          
cui-submit폼 제출 버튼 지정

          
cui-ignore폼 모듈 렌더링 제외 (네이티브 스타일 사용 시)

          
cui-skin체크박스/라디오 스킨 (primary, switch)

          
cui-text스위치 ON/OFF 텍스트 (예: "열기|닫기")

          
title체크박스/라디오 라벨 텍스트

            규칙
            설명
