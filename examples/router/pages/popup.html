기본 팝업 (Popup)

홈 팝업예제 기본팝업
Alert / Confirm / Prompt popup.alert(), popup.confirm(), popup.prompt()

기본적인 대화상자 팝업입니다.

메시지 & 로딩 popup.msg(), popup.load()

간단한 메시지와 로딩 인디케이터입니다.

토스트 (Toast) popup.toast()

화면 모서리에 표시되는 알림 메시지입니다.

툴팁 (Tips) popup.tips()

특정 요소에 붙는 말풍선 팁입니다.

팝업 메서드 요약
메서드 설명
popup.alert(content)알림 팝업
popup.confirm(content, yes)확인/취소 팝업
popup.prompt(options, yes)입력 팝업
popup.msg(content, options)메시지 표시 (자동 닫힘)
popup.load(icon)로딩 인디케이터
popup.toast(options)토스트 알림
popup.tips(content, elem)툴팁
popup.close(index)팝업 닫기
